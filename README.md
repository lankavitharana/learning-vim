# learning-vim

## Delete without copying to the register
`_dd`

## Verbs in vim
* `d` delete
* `c` change

## Modifiers in vim
* `t` till
* `f` find
* `i` inside

## Objects in vim
* `w` word
* `(` parenthesis
* `t` tag
* `s` sentence
* `"` quotes
## Navigation

* Move to the other bracket easily using `%` 
* Go back to the previos location using `Ctrl + O`
* Go forward to the forward location using `Ctrl + I`
